# Petstore API Test

Este proyecto contiene pruebas automatizadas para la API de Petstore utilizando RestAssured y JUnit. Las pruebas cubren operaciones básicas de CRUD (Create, Read, Update, Delete) sobre la entidad Pet.

## Requisitos

- Java 15 o superior
